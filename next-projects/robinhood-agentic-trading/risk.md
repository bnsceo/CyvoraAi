# Risk

This project should be treated as a controlled experiment, not as a shortcut to profit.

## Hard Rules

- No margin.
- No options.
- No leverage.
- No unattended order placement at the start.
- No trading money needed for bills or essentials.
- No strategy changes without logging the reason.

## Position Sizing

- Start with a tiny dollar amount.
- Set a maximum per-trade allocation.
- Set a maximum daily loss.
- Set a maximum number of trades per day.

## Review Rules

- Recheck every trade before it is sent.
- Record the AI reasoning.
- Record the human approval or rejection.
- Stop trading if the agent starts producing unclear or inconsistent ideas.

## Failure Modes To Watch

- Chasing losses
- Overtrading
- Ignoring fees or spreads
- Hallucinated market facts
- Unclear prompts that create inconsistent behavior

Runtime modes and roadmap

Built progressively, without pretending unfinished systems are complete.

Persistent runtime / memory / consensus. Cyvora should not be just a session-by-session app. The next milestone is durable state, background maintenance, and a controlled review path for anything that becomes persistent.
